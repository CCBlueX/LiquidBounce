How to install LiquidBounce for Minecraft {mcversion}
  1. Download and install Forge for Minecraft {mcversion} from minecraftforge.net.
  2. Open the .minecraft folder.
  3. Find a folder called mods (If it doesn't exists, you should create new one).
  4. Copy this file{filename} into it.
  5. Launch Minecraft with the Forge profile selected.
For more detailed instructions visit liquidbounce.net
